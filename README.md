# Recommendation-algorithm
A-level programming project. Item-based collaborative filtering algorithm + data.
Recommendation algorithm completed in final year of A-levels (2023).

Uses customer purchase histories and product ratings to determine which products should be recommened to a customer.
Also contains a basic user interface created using TKinter, and very basic inventory forcasting algorithm.
Created in Python.
Libraries user: TKinter, datetime, pandas.
Data read from CSV file.
